# Download the MediaPipe FaceLandmarker Model

Place the model file `face_landmarker.task` in this directory.

Download from:
https://storage.googleapis.com/mediapipe-models/face_landmarker/face_landmarker/float16/1/face_landmarker.task

File size: ~3.4 MB

Without this file, the MediaPipe face landmark detection will not work.
The app will show "Model file missing" and fall back gracefully.

DO NOT commit the .task file to git — add it to .gitignore.
